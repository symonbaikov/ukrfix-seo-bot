# UkrFix SEO Bot — New Features

## SEO Content & Structure
- **Title optimization**: AI returns raw title, then we normalize length, remove stop-words, and keep title/sentence case variants (`src/utils/title_optimizer.py`, used in `src/seo_generator.py`).
- **Meta description**: 140–160 chars with CTR cues (⭐ найкраще, швидко, корисно) auto-filled when missing (`src/seo_generator.py`).
- **Slug generation**: Takes H1, auto-translates to English (fallback transliteration), drops stop-words/symbols, keeps 3–5 keywords, avoids template slugs, 75-char cap (`src/utils/slugify.py`).
- **Rich HTML**: Prompts enforce h1/h2/h3, ul/li, short paragraphs, sample ad snippet, and auto-append CTA block (`src/utils/content_enhancer.py`).

## Taxonomy & Publishing
- **Tags (5–8) + category** are generated by AI, cleaned, and pushed to WP; missing terms are created automatically (`src/wp_publisher.py`).
- **Excerpt/meta**: Meta description is sent as WordPress excerpt; custom slug is published too (`src/wp_publisher.py`).

## Uniqueness & Internal Linking
- **Uniqueness guard**: Keeps `data/published_articles.json` of titles/slugs; skips duplicates before publishing (`src/utils/history.py`, used in `src/bot.py`).
- **Internal link builder**: Picks 1–2 similar past posts (by tags/title/category) and injects a “Читайте також” block before CTA (`src/utils/content_enhancer.py`, `src/utils/history.py`).

## Media Improvements
- **Pexels relevancy**: Fetches up to 8 photos, scores by alt keyword match, prefers landscape, and uses optimized sizes (`src/image_service.py`).
- **Featured image**: Uploaded to WP and attached as `featured_media` when available (`src/image_service.py`, `src/wp_publisher.py`).

## Generation Flow
- **Prompted JSON**: Gemini returns JSON with title, title_sentence, meta_description, tags, category, and HTML content; cleaned of Markdown fences (`src/seo_generator.py`).
- **Test mode parity**: Draft runner now uses the same pipeline (slug/tags/category/links), with test prefixes to avoid collisions (`run_test_mode.py`).

## Dependencies
- Added transliteration dependency: `cyrtranslit>=1.1.0` (`requirements.txt`).
